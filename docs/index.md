# Missouri Class F Driver License Study Workbook

Welcome to the MkDocs site version of the study workbook. This site paraphrases core safe-driving and licensing principles for Missouri's Class F (non-commercial) driver license exam preparation. Always consult the official Missouri Driver Guide for authoritative details.

## Using This Site
1. Start with the Outline for module objectives.
2. Read each module and apply active recall.
3. Take the related quiz immediately after studying.
4. Review answers and explanations.
5. After all modules are complete, attempt the practice exam (coming soon).

## Scope
- Focus: Class F (standard non-commercial) driving knowledge.
- Excludes: CDL-specific material and endorsements.
- Emphasis: Rules of road, signs, safe operation, responsibility, impairment, point system.

## Disclaimer
This is not an official publication and omits verbatim state text; numbers (distances, fines, BAC thresholds) must be verified in the official guide.

Use the left sidebar or top nav (depending on theme) to jump between modules, quizzes, and answer keys.
